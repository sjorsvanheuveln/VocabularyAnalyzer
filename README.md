# VocabularyAnalyzer

## Nederlands
Eenvoudig R script dat meest voorkomende lettercombinaties in de Nederlandse taal berekent aan de hand van een vocabulaire (text file).
Script is makkelijk aan te passen om andere analyzes uit te voeren.

## English
Simple R script that counts occurrences of letter combinations in the Dutch language. Can easily be modified to analyze other languages.
E.g for english vocabulary:
'''
data <- fread('https://raw.githubusercontent.com/titoBouzout/Dictionaries/master/English.dic')
'''
